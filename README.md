## Stack utilizada

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/leokazuyukinagatani/nlw-copa-mobile
```

Entre no diretório do projeto

```bash
  cd nlw-copa-mobile
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  expo start
```
